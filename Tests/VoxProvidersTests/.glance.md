## Purpose
This directory contains unit and integration tests for audio conversion utilities and various Speech-to-Text (STT) and text-rewrite service providers. It validates the implementation of client-side logic for Deepgram, ElevenLabs, Gemini, InceptionLabs, and OpenRouter, ensuring correct request formatting, error mapping, and session management. The suite also verifies local audio processing capabilities and the robust handling of network timeouts and provider fallback chains.

## Key Roles
- **Client Tests (`Deepgram...`, `ElevenLabs...`, `Gemini...`, `InceptionLabs...`, `OpenRouter...`)**: Verify REST and WebSocket communication, including multipart MIME type mapping, JSON payload structure, and the translation of HTTP status codes into domain-specific errors.
- **AudioConverter Tests**: Validate CAF to WAV and Opus/OGG conversion logic, including checks for system-level executable availability for `afconvert`.
- **ProviderAssemblyTests**: Ensures the correct construction of provider fallback chains, retry logic, and the wiring of instrumentation hooks.
- **Timeout Management Tests (`PhaseAwareSTT...`, `BatchSTTTimeouts...`)**: Test the state machine and scaling logic used to manage separate timeouts for the upload and processing phases of STT requests.
- **SpeechTranscriberClientTests**: Verifies the API contract and availability gating for Apple's Foundation-based speech transcription.
- **URLProtocolStub**: Provides a shared utility for intercepting network requests and providing mocked responses to isolate client logic from live services.

## Dependencies and Caveats
- Requires `/usr/bin/afconvert` for certain local audio conversion tests.
- Uses `ffmpeg` (expected at `/opt/homebrew/bin/ffmpeg`) for generating silent test audio files in specific integration scenarios.
- `SpeechTranscriberClientTests` are gated by a minimum requirement of macOS 26.0.
- Integration tests for Deepgram require a valid `DEEPGRAM_API_KEY` environment variable; otherwise, they are skipped.
- Relies on `URLProtocol` subclassing for network isolation and payload inspection.
- Timeout calculations are dependent on file size, using a base duration plus a per-megabyte scaling factor.
- Temporary file system access is required for reading and writing test audio buffers (CAF, WAV, OGG).
