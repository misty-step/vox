## Purpose
This directory provides the core abstractions and infrastructure for a dictation and text-processing pipeline. It defines protocols for speech-to-text (STT) and text rewriting, alongside decorators for handling retries, fallbacks, concurrency limits, and health-based routing. The module also includes utility types for error handling, multipart data construction, and quality evaluation of processed text.

## Key Roles
- `Protocols.swift`: Defines the central interfaces for STT providers, streaming sessions, text rewriting, and audio recording.
- **STT Provider Decorators**: Files including `RetryingSTTProvider`, `FallbackSTTProvider`, `HedgedSTTProvider`, and `HealthAwareSTTProvider` provide execution strategies for transcription tasks.
- `ModelRoutedRewriteProvider.swift`: Directs text-processing requests to specific clients (Gemini, Inception, or OpenRouter) based on model string prefixes.
- `RewriteQualityGate.swift`: Computes similarity scores and word overlap to determine if a rewritten transcript meets defined acceptability thresholds.
- `Errors.swift`: Centralizes domain-specific errors and provides logic for retryability and fallback eligibility.
- `MultipartFormData.swift`: Facilitates the construction of multipart requests, including a streaming builder to minimize memory usage for audio files.
- `ProcessingLevel.swift`: Categorizes dictation processing into `raw`, `clean`, or `polish` modes with associated default models.

## Dependencies and Caveats
- `HealthAwareSTTProvider` uses an internal `HealthStore` to rank providers by success rate and average latency.
- `AsyncSemaphore` manages concurrency using `NSLock` and `CheckedContinuation` for thread-safe task suspension.
- `SecureFileDeleter` notes that secure overwriting is ineffective on APFS due to copy-on-write semantics.
- `TimeoutSTTProvider` supports dynamic timeouts that scale based on the file size in megabytes.
- `RewriteQualityGate` utilizes a normalized Levenshtein distance and a filtered content-word overlap score.
- `ModelRoutedRewriteProvider` implements specific routing lanes for `gemini-`, `google/gemini-`, and `mercury-` model prefixes.
- `HedgedSTTProvider` cancels all outstanding tasks in its task group as soon as the first successful transcription is returned.
- `STTError` includes logic to distinguish between transient failures and permanent failures for health scoring purposes.
