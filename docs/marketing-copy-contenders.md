## Copy Rules

### Do NOT include:
- "Join private beta" (no beta exists)
- Privacy claims (we send data to third parties)
- "Why it feels different" (overcomplicated)
- vox.app domain (doesn't exist)
- "No accounts" / "No subscriptions" (we need both)

### DO include:
- Contact: hello@mistystep.io
- Link to mistystep.io
- "A MistyStep project" in footer
- Download for macOS button
- Simple value prop: talk → transcribe → optional cleanup → clipboard
