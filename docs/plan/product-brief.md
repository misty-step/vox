# Product Brief

## Thesis
- Invisible editor: spoken intent → clean text at cursor
- No commands, no modes, no settings maze
- One optional setting: processing/rewrite level

## Core loop
- Tap hotkey to start listening
- Speak
- Tap hotkey to submit → commit STT → rewrite → insert

## User promise
- Faithful meaning, no invented content
- Quality > speed for long dictations
- Works everywhere, no app integrations

## Guardrails
- No semantic hallucinations
- No secrets in client
- Minimal UI, glyph only

## Target platform
- macOS first, menu bar app

## MVP scope
- Realtime STT
- Rewrite via LLM
- Paste into active app
- Context.md support
